HR Employee Retention Analytics Dashboard
!

📋 Project Overview
This project focuses on analyzing employee turnover and identifying key factors that contribute to employee attrition. Using Power BI, I transformed raw HR data into an interactive dashboard that allows stakeholders to visualize trends, pinpoint high-risk departments, and develop data-driven retention strategies.

🎯 Business Objectives
Identify Attrition Trends: Track the overall turnover rate across the organization.

Demographic Analysis: Understand how age, gender, and marital status correlate with retention.

Performance vs. Satisfaction: Analyze the relationship between monthly income, job satisfaction scores, and the likelihood of leaving.

Departmental Deep-Dive: Pinpoint specific departments (e.g., Sales, R&D) with higher-than-average resignation rates.

🛠️ Tech Stack & Tools
Data Source: CSV/Excel (HR Dataset)

Analysis Tool: Power BI Desktop

Data Transformation: Power Query (DAX for calculated measures)

Visualization: Donut Charts, Bar Charts, Treemaps, and KPI Cards

📊 Key Insights & Features
The dashboard provides a comprehensive view of the following metrics:

Total Employee Count & Active Employees.

Attrition Rate (%): The percentage of employees who have left the company.

Average Age: Understanding the lifecycle stage of the workforce.

Attrition by Salary Hike: Determining if financial growth directly impacts loyalty.

Job Role Satisfaction: A heatmap or bar chart showing which roles report the lowest satisfaction.
